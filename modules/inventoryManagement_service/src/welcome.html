Welcome to salesai-inventoryManagement Service

Maintains inventory records per store, tracks levels and item status, logs all updates and movements (including sales-driven and manual adjustments), detects and publishes low-stock alerts, provides inventory analytics at store and manager level, and features Redis caching for fast lookups.

Available Routes

  • getInventoryItem => GET /inventoryitems/:inventoryItemId
  • createInventoryItem => POST /inventoryitems
  • updateInventoryItem => PATCH /inventoryitems/:inventoryItemId
  • deleteInventoryItem => DELETE /inventoryitems/:inventoryItemId
  • listInventoryItems => GET /inventoryitems
  • getInventoryMovement => GET /inventorymovements/:inventoryMovementId
  • createInventoryMovement => POST /inventorymovements
  • deleteInventoryMovement => DELETE /inventorymovements/:inventoryMovementId
  • listInventoryMovements => GET /inventorymovements
  • getLowStockAlert => GET /lowstockalerts/:lowStockAlertId
  • createLowStockAlert => POST /lowstockalerts
  • resolveLowStockAlert => PATCH /resolvelowstockalert/:lowStockAlertId
  • deleteLowStockAlert => DELETE /lowstockalerts/:lowStockAlertId
  • listLowStockAlerts => GET /lowstockalerts

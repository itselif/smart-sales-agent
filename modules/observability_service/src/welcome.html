Welcome to salesai-observability Service

Collects, indexes, and returns audit logs for all sensitive actions and system metrics, supporting audit trail retrieval by user, store, action, or time. Stores and serves system/business metrics, exposes endpoints for audit export, anomaly events, and notification flows.

Available Routes

  • getAuditLog => GET /auditlogs/:auditLogId
  • createAuditLog => POST /auditlogs
  • updateAuditLog => PATCH /auditlogs/:auditLogId
  • deleteAuditLog => DELETE /auditlogs/:auditLogId
  • listAuditLogs => GET /auditlogs
  • getMetricDatapoint => GET /metricdatapoints/:metricDatapointId
  • createMetricDatapoint => POST /metricdatapoints
  • updateMetricDatapoint => PATCH /metricdatapoints/:metricDatapointId
  • deleteMetricDatapoint => DELETE /metricdatapoints/:metricDatapointId
  • listMetricDatapoints => GET /metricdatapoints
  • getAnomalyEvent => GET /anomalyevents/:anomalyEventId
  • createAnomalyEvent => POST /anomalyevents
  • updateAnomalyEvent => PATCH /anomalyevents/:anomalyEventId
  • deleteAnomalyEvent => DELETE /anomalyevents/:anomalyEventId
  • listAnomalyEvents => GET /anomalyevents

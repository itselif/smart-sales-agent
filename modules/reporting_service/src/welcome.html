Welcome to salesai-reporting Service

Generates downloadable daily sales and analytics reports (per store, user, or cross-store/by parameter), issues signed URLs for secure, time-limited access, audits all generation/download activities, and allows admins to configure reporting policies.

Available Routes

  • getReportRequest => GET /reportrequests/:reportRequestId
  • createReportRequest => POST /reportrequests
  • updateReportRequest => PATCH /reportrequests/:reportRequestId
  • deleteReportRequest => DELETE /reportrequests/:reportRequestId
  • listReportRequests => GET /reportrequests
  • getReportFile => GET /reportfiles/:reportFileId
  • createReportFile => POST /reportfiles
  • updateReportFile => PATCH /reportfiles/:reportFileId
  • deleteReportFile => DELETE /reportfiles/:reportFileId
  • listReportFiles => GET /reportfiles
  • getReportPolicy => GET /reportpolicies/:reportPolicyId
  • createReportPolicy => POST /reportpolicies
  • updateReportPolicy => PATCH /reportpolicies/:reportPolicyId
  • deleteReportPolicy => DELETE /reportpolicies/:reportPolicyId
  • listReportPolicies => GET /reportpolicies

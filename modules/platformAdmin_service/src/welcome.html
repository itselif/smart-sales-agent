Welcome to salesai-platformAdmin Service

Platform-wide backend administration: OpenAPI schema/versioning, system health checks, CI/CD, repo/cache backend switching, seed/test data operations, JWT/session config & rotation, API/WebSocket auth monitoring, global config control. Coordinates and logs all admin operations; RBAC and stores managed in auth/store services.

Available Routes

  • getOpenApiSchema => GET /openapischemas/:openApiSchemaId
  • createOpenApiSchema => POST /openapischemas
  • updateOpenApiSchema => PATCH /openapischemas/:openApiSchemaId
  • deleteOpenApiSchema => DELETE /openapischemas/:openApiSchemaId
  • listOpenApiSchemas => GET /openapischemas
